Question-1: What are the different ways to select an element in the DOM?

We can select an element by using :
1. getElementById()-select by id.
2. getElementsByClassName() -givers a collection of elements.
3. getElementByTagName() -gives a collection of element.
4.querySelectorAll() and querySelector().

Question-2: What is the difference between innerHTML, innerText, and textContent ?

1. innerHTML bring us html content of a element.
2. innerText brings us text content of a element.
3. textContent brings us text content as well as hidden text or style.

Question-3: What is event delegation in the DOM?

Event delegation used to handle multiple child elements with a single event listener on the parent. It is efficient than adding multiple event listener in child element.

Question-4: What is event bubbling in the DOM?

When we click on a child element (like a button inside a div), the event does not stop there. It moves up to parent (after parent it also goes to grandparent as well) elements around it (like the div and then the body).
